# StyleSphere Analytics

### Fashion E-Commerce Sales & Demand Forecasting Platform

## Overview

StyleSphere Analytics is an end-to-end analytics project designed to analyze customer purchasing behavior, revenue trends, category performance, return patterns, and forecast monthly sales for an Indian fashion e-commerce marketplace.

## Tech Stack

* MySQL
* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Power BI

## Key Features

* Revenue & KPI reporting
* Customer segmentation analysis
* Category and product performance insights
* Return analysis
* Monthly sales forecasting
* Executive dashboard reporting

## Business Problem Solved

This project helps improve:

* inventory planning
* customer targeting
* discount optimization
* return reduction
* sales forecasting

## Project Modules

1. SQL Analytics
2. Python EDA
3. Machine Learning Forecasting
4. Power BI Dashboard
